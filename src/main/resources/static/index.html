Guide
Managing cans
  1. Add (top right) → fill at least the SKU → Save
  2. Edit → open a can detail → click Edit
  3. Delete → open Edit form → click Delete (confirmation required)
Photos
  1. Click a slot to pick a file, or drag & drop an image onto it
  2. Click 🔗 to paste a direct image URL instead of uploading
Import & Export
  1. Export → downloads the full collection as an Excel file
  2. Import → upload that Excel file: new rows are added, existing IDs updated, photos untouched
Condition values: OK · MINOR DENTS · DAMAGED
Browse & Filter
  1. Type in the search box to find cans by name or SKU
  2. Use the dropdowns to filter by Country, Size, Manufacturer, or Tab type
  3. Chips: Promo (limited editions) · FULL (sealed) · With/No photo
Compare
  1. Click any can to open its detail view
  2. Click Compare in the detail panel — a bar slides up at the bottom
  3. Add up to 3 cans, then click Compare to see them side by side
Conditions & FULL
  1. OK — perfect condition
  2. MINOR DENTS — small dents, still collectible
  3. DAMAGED — visible damage or deformation
  4. FULL badge — the can is sealed and still contains liquid
